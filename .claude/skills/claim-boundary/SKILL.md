# Claim Boundary

Use whenever writing UI text, README, exports, or demo narration. Runtime
implementation: `src/core/claimFirewall.ts`.

## Block

discovers quantum biosensors; predicts magnetic response; validated sensor;
working construct; ready-to-test sequence; Nebula ranking; Astra score;
sequence alone predicts spin response; synthetic traces validate biology.

## Prefer

public construct hypothesis; measurement-worthiness; source-backed rationale;
synthetic assumption sweep; requires measurement; collaborator handoff.

## Rewrite pattern

Unsafe: "This construct is a quantum biosensor."
Safe: "This public construct hypothesis has a plausible mechanism route worth
measuring under explicit assumptions."
