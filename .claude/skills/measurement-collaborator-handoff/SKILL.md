# Measurement Collaborator Handoff

## Input

Download Markdown handoff from the Measure next screen (selected hypothesis).

## Output structure (1 page)

1. **Objective** (2 sentences)
2. **Measure first:** top construct + route name
3. **Primary readout + null expectation**
4. **3 required controls** (from export)
5. **Falsification kill rules** (from export)
6. **Confounders to watch**
7. **Allowed claim** (one sentence, claim-safe)
8. **Not allowed** (no validation, no prediction)

## Claim boundary

Handoff is triage for planning — not an experimental protocol or validated design.
