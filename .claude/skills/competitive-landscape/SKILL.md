# Competitive Landscape

## Honest positioning

| Tool | What it does | What Discover adds |
| --- | --- | --- |
| AlphaFold DB | Structure | Mechanism route + measurement triage |
| FPbase | FP catalog | Multimodal sweep + controls |
| ESM / embeddings | Sequence representation | Public analog search only — no spin claim |
| RFdiffusion / MPNN | Design | Downstream handoff stub — not discovery engine |
| Generic LLM chat | Text | Pipeline + firewall + falsification + swarm |

## Say

"Complementary workflow layer — decides what deserves measurement first."

## Never say

"We replace AlphaFold" or "we predict sensor response."
